# Personal Academic Portfolio & Student Management Platform

### Project Overview
This project is a fully functional, responsive multi-page web application designed as a personal academic portfolio and student management system. The platform is engineered to showcase student profiles, technical skills, academic histories, and simulated development projects, while offering interactive utilities for managing real-time academic tasks.

### Core Architecture & File Mapping
The application is built using modular web standards with the following structural layout:
*   **Homepage:** Houses the core student biography, profile context, and primary navigation wrapper.
*   **About Me Page:** Displays educational timelines using structured tables, technical skill inventories, and a integrated HTML5 multimedia audio environment.
*   **Projects Page:** Utilizes a responsive CSS Grid system to display simulated engineering project cards linked to active developer platforms.
*   **Academic Planner Page:** Features an interactive, array-driven task management system utilizing DOM manipulation for creating, completing, and deleting tasks.
*   **Contact Page:** Features a secured client-side form backed by structural JavaScript regular expression (Regex) validation checks.

### Technical Implementations
*   **Structure:** Pure semantic HTML5 (Tables, Lists, Forms, Multimedia, and Hyperlinks).
*   **Styling:** External custom CSS stylesheet utilizing Flexbox, CSS Grid layouts, media queries for mobile responsiveness, and interactive hover transitions.
*   **Logic:** Vanilla JavaScript managing state arrays, events, runtime rendering, and input data validation.
